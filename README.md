# aplicaciones-web-y-moviles

Repositorio oficial de la asignatura **Aplicaciones Web y Móviles**

---
## Descripción
En esta asignatura los estudiantes desarrollarán habilidades para el diseño y construcción de aplicaciones web
El enfoque de la materia está basada en:

- Desarrollo Practico de software
- Contrucción progresiva de aplicaciones reales
- Trabajo colaborativo y resolución de problema
